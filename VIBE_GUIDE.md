# AI Personal Trainer: Vibe Coding Guide

Welcome to the **AI Personal Trainer** project. This repo is designed to be "vibe-coded" — optimized for collaboration between AI agents and human coaches/developers.

## The Workflow

### 1. The Source of Truth: `VIBE_GUIDE.md`
Always update this file if the trainer's "coaching style" or technical direction shifts.

### 2. Context Syncing (`/prompts`)
Keep the "personality" of the trainer consistent.
- **Coach Persona**: How does the assistant talk? (Encouraging, Drill Sergeant, etc.)
- **Vision Logic**: High-level descriptions of how to identify a "good rep".

### 3. Agent-Friendly Commits
Explain the *intent*.
Example: `feat: add squat form detection vibe using MediaPipe`

### 4. Collaboration with Friends
1. **Invite your friend** as a collaborator on GitHub.
2. **Vibe Requests**: Use GitHub Issues to propose new exercises or coaching features.

## Tech Stack
- **Vision**: MediaPipe (Pose), OpenCV.
- **Brain**: Agentic fitness coaching logic.
- **UI**: Real-time feedback overlay.
